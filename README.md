## 项目硬件架构

整套设备分为两块独立电路板：

### 1. 飞行器飞控主板
| 模块 | 硬件 | 功能 |
|---|---|---|
| MCU | STM32F103C8T6 | 主控 |
| IMU | MPU6050 六轴传感器 | 姿态解算 |
| 无线 | NRF24L01 2.4G | 接收遥控指令 |
| 驱动 | 4路空心杯电机 | PWM 调速 |
| 存储 | AT24C02 | PID 参数掉电保存 |
| 外设 | 蜂鸣器、8路按键 | 故障报警、模式切换 |

### 2. 无线遥控器
| 模块 | 硬件 | 功能 |
|---|---|---|
| MCU | STM32F103C8T6 | 主控 |
| 输入 | 双模拟摇杆 + 8路独立按键 | 油门/俯仰/横滚/偏航输入 |
| 无线 | NRF24L01 2.4G | 发送控制指令 |
| 外设 | 蜂鸣器 | 操作反馈 |

## 开发方式

全程**寄存器直接操作**，不依赖 HAL 库：
- GPIO：CRL/CRH/ODR 寄存器直操
- I2C：MPU6050 姿态采集、AT24C02 参数存储
- SPI：NRF24L01 双向无线通信
- TIM：4路 PWM 电机调速
- ADC：摇杆模拟量采集
- 软件毫秒延时
